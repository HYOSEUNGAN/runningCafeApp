## 🎨 디자인 시스템 적용

### 컬러 스킴
```css
Primary: #A259FF ~ #6A82FB (그라데이션)
Secondary: #FFB86C (오렌지), #43E97B (민트)
Neutral: #F5F6FA (배경), #22223B (텍스트)
System: #FF4D4F (에러), #43E97B (성공)
```

### 타이포그래피
```css
H1: 32px (페이지 타이틀)
H2: 28px (섹션 헤딩)  
H3: 24px (카드 타이틀)
Body: 16px (본문)
Caption: 13px (부가 정보)
```

### 컴포넌트 스타일
- **카드**: 둥근 모서리 (12px), 그림자 효과
- **버튼**: 그라데이션 배경, 호버 효과
- **입력 필드**: 포커스 시 보라색 강조
- **아이콘**: 듀오톤 스타일, 러닝/카페 테마

---

## 📱 반응형 디자인

### 브레이크포인트
- **모바일**: < 768px
- **태블릿**: 768px - 1024px  
- **데스크톱**: > 1024px

### 모바일 최적화
- 터치 친화적 버튼 크기 (최소 44px)
- 스와이프 제스처 지원
- 하단 네비게이션 (엄지손가락 영역)
- 한 손 조작 고려한 UI 배치
